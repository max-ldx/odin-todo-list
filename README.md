# odin-todo-list

This is a todo list app. This project is part of [The Odin Project](https://www.theodinproject.com/lessons/node-path-javascript-todo-list)

## Technologies

- HTML
- CSS
- JavaScript
- localStorage
- Webpack

## Goal

Build a fully functional todo list app with single page app behavior. This project focuses on aggregating all the knowledge gained so far into a single practice unit and using localStorage.

## Learning outcomes

- A solid refresher on HTML, CSS, JS, DOM manipulation, Webpack
- Using localStorage for the first time
- Applying good code design (SOLID, DRY, ...)

## How it's built

The app is built from a single HTML file and many JS files.
The files are bundled together with Webpack to optimize development and production.
